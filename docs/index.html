✏️ Edit Treatment Logic

The treatment_logic placeholder inserts different prompts based on descriptor type:
  • Low-weight descriptors (weight < 8): Uses "Treatment Response" prompt
  • High-weight descriptors (weight ≥ 8): Uses "Intention to Treat" prompt
Weight < 8
Used for: arthritis, myositis, rash, alopecia, mucosal ulcers, pleurisy, pericarditis, low_complement, increased_DNA_binding, fever, thrombocytopenia, leukopenia
Weight ≥ 8
Used for: seizure, psychosis, organic brain syndrome, visual disturbance, cranial nerve disorder, lupus headache, CVA, vasculitis

✏️ Edit NPSLE Tips

These tips are added to neuropsychiatric SLE descriptors (seizure, psychosis, organic brain syndrome, visual disturbance, cranial nerve disorder, lupus headache, CVA, vasculitis)
These tips help the LLM better understand neuropsychiatric manifestations in SLE
